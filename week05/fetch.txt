FETCH
********
- Used for making requests from servers
- Retrieve data from APIs
- Interact with servers

Rest Parameters
***************
- Denoted with '...'
- If at end of function parameters, it gathers the rest of the list of arguments into an array
- If occurs in function call, it's a 'spread syntax' and expands array into a list
